# macOS Productivity Shortcuts

## AeroSpace (Alt = modifier, vim-style)

### Focus & Move
| Action | Shortcut |
|---|---|
| Focus left/down/up/right | Alt + H/J/K/L |
| Move window left/down/up/right | Alt+Shift + H/J/K/L |
| Cycle next window | Alt+N |
| Cycle prev window | Alt+Shift+N |

### Workspaces
| Action | Shortcut |
|---|---|
| Switch to workspace 1-9 | Alt + 1-9 |
| Move window to workspace 1-9 | Alt+Shift + 1-9 |
| Move window + follow to workspace | Alt+Ctrl + 1-9 |

**Workspace → monitor assignment:**
- 1, 2, 3 → Built-in Retina Display
- 4, 5, 6 → Samsung Odyssey G8 (fallback: 2nd monitor / secondary)
- 7, 8, 9 → LG IPS QHD (fallback: 3rd monitor)

### Monitors
| Action | Shortcut |
|---|---|
| Focus next/prev monitor | Alt+Tab / Alt+Shift+Tab |
| Move to monitor + follow | Alt+Ctrl + H/J/K/L |

### Layout
| Action | Shortcut |
|---|---|
| Toggle tile layout | Alt+/ |
| Accordion layout | Alt+, |
| Horizontal tiles | Alt+E |
| Vertical tiles | Alt+S |
| Fullscreen | Alt+F |
| Float toggle | Alt+Shift+F |
| Resize shrink/grow | Alt+Shift + -/= |

### Service
| Action | Shortcut |
|---|---|
| Enter service mode | Alt+Shift+; |
| Reload config (in service mode) | Esc |

## Zsh (Terminal)

| Action | Shortcut |
|---|---|

|---|---|
